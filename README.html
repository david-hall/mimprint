Introduction

Jiviewer is an image viewing and printing program written in Java and distributed under the GPL.

Home Page

Go to the Jiviewer home page to download the latest version of Jiviewer.

Features

  • Licensed under GPL, freely redistributable with source.
  • Written in Java, runs on Windows, Mac and Unix.
  • Four viewing modes:
    • Normal viewing mode allows easily stepping through a folder of image files.
    • Full screen mode scales images to fit in a full-window frame.
    • Alternate screen mode displays images on a second screen.
    • Printable mode allows collecting images onto a page for printing.
  • Browser with icons allows easy navigation of image directories.
  • Browser allows easy navigation through collections of layout templates.
  • Images and directories can be annotated using a text file.
  • Easy custom layout of printable pages.
  • Load/save custom layout templates.

System Requirements

Item Required Why
Processor 700 MHz Image rendering will be somewhat slow with older processors. Faster processors will of course give better results.
Memory 256 MB Graphical Java programs tend to use a lot of memory. Images take up a lot of memory.
Disk 1 MB,
plus up to 250 MB
to install Java		 If you already have Java installed, installing Jiviewer will take less than 1 MB. Installing Java takes 100 MB or more, depending on which version you install, and whether or not you install the documentation.
Java JRE 1.5 Jiviewer uses Drag and Drop.
  • On Linux, Java 1.5.0 or better is required; bugs in 1.4 cause drag and drop not to work, which makes the Printable page unusable.
  • On Mac, Java 1.4.2 or better is required.
  • On Windows, Java 1.4.2 or better is required. Unfortunately, dragging from the browsing list is not working on Windows; however, you can drag from Windows Explorer and drop into the Printable page.

Installing Java

To install Java on your system, download JRE1.4 from Sun's Java web site and follow those installation directions. When you are at that download page, look for the section that says Download J2SE v 1.4.2_03 (or possibly _04 or later), find your platform, and select the DOWNLOAD link from the JRE column.

Running Jiviewer

If you have not already done so, you must install Java.

On Windows

On Windows, double-click on the shortcut (if you have set one up) or CD to the directory containing jiviewer.jar and run

java -jar jiviewer.jar
You can manually set up a shortcut to jiviewer.jar.

On Unix

On Unix, run
java -jar jiviewer.jar
If the java binary is not in your PATH, you can explicitly specify the path to it. For example, if you have installed JDK1.4 into /usr/local/j2sdk1.4.2, you can run jiviewer with the command
/usr/local/j2sdk1.4.2/bin/java -jar jiviewer.jar

On Solaris 2.6 or later, you can directly run jiviewer.jar.
